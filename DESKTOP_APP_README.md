- PyQtChart installed: `pip install PyQtChart`

## 🚨 Troubleshooting

### Window doesn't open
```bash
pip install PyQt5 PyQtChart --upgrade
python main.py
```

### Import errors
```bash
python launch_verify.py  # Diagnose
pip install -r requirements.txt  # Fix
```

### Still having issues
```bash
# Use CLI version instead
python cli.py
```
